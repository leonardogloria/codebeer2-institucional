Docker & AWS

Bate Papo sobre Docker e Servless Com AWS

  • 3 de Julho,2017
  • Wells Beer

Descrição

Salve galera! Iniciaremos nossos trabalhos etílicos com duas palestras sobre temas que estão com muito Hype. Vamos descobrir em que o Docker pode nos ajudar a criar ambientes de desenvolvimento com altissíma produtividade e aprenderemos o que é orquestrar um container! Logo em seguida falaremos sobre como desenvolver uma aplicação complexa com banco de dados NoSql,NodeJS , autenticação, envio de email, estatísticas sem precisarmos de nenhum servidor! E com bastante Cerveja!
